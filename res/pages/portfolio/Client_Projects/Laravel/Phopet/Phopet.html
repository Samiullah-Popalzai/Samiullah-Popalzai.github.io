Phopet is built on a robust Laravel PHP backend running on Nginx server for high-performance request handling. The platform leverages Laravel's queue system for asynchronous AI model training (30-60 minutes), Eloquent ORM for user and credit management, and Redis for caching. JavaScript powers the dynamic upload interface, real-time training progress indicators, and interactive album selections. The AI image generation integrates with custom-trained models via secure API endpoints.

My Role

As the lead backend and full-stack developer for Phopet, I architected and built the entire Laravel application from the ground up. I designed the database schema for user accounts, credit systems, model training jobs, and generated image storage. I implemented the queue worker infrastructure to handle long-running AI training tasks without blocking the user interface, ensuring a smooth experience even during peak loads.

I developed RESTful APIs for communication between the frontend JavaScript and the AI generation backend, integrated payment processing for one-time credit purchases, and built admin dashboards for monitoring system health. I also implemented automated cleanup jobs for inactive user data (60-90 days), email notifications via Laravel notifications, and optimized Nginx configuration for static asset delivery of generated pet images. My role covered backend architecture, API development, database design, AI service integration, and performance optimization.

Phopet is an AI-powered platform that transforms ordinary pet photos into stunning, studio-quality artwork. Built exclusively for pets, the platform allows users to upload 10-20 photos of their furry friends to train a custom AI model. Once trained, users can generate images from over 40 themed albums (Superheroes, Travel, Studio, Dystopian) and even upscale them for high-resolution prints.

The client needed a scalable, high-performance web application that could handle asynchronous AI model training, credit-based payments, and a seamless user experience. I built the platform using Laravel on Nginx, with a JavaScript frontend for dynamic interactions. The backend manages user authentication, credit purchases, queue jobs for AI training, and secure API integrations with the image generation service.

One key challenge was handling the 30-60 minute AI training time without forcing users to wait on a loading screen. I implemented Laravel's queue system with Redis to process training jobs asynchronously, sending email notifications via Laravel Notifications when a model is ready. Another challenge was data cleanup — I built automated scheduled commands to permanently delete inactive user models and images after 90 days, keeping the database lean and compliant with privacy expectations.

After launch, Phopet successfully processes hundreds of concurrent training jobs, with users receiving their generated images reliably. The one-time payment model (starting at €9.90 for 50 images) has proven attractive to pet owners who want quality results without subscription fatigue. The platform continues to scale, with additional albums and features being added regularly through the Laravel-based admin dashboard.

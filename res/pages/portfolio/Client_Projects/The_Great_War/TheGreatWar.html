The Great War

The first FPS shooting game from Afghanistan, optimized for mobile devices. Built for fun and learning.

Project Overview

The Great War is a mobile FPS game in development using Unity. Players enjoy touch-based shooting controls, aiming, and tactical movement on mobile devices. This solo project is fully developed by me and aims to be the first Afghan FPS game playable on smartphones and tablets.

Mobile Demo (Coming Soon)

Your Role

I am the sole developer responsible for coding all gameplay mechanics, implementing touch controls, designing mobile-friendly levels, creating assets, and optimizing performance for Android and iOS devices.

Features

  • Mobile FPS controls with touch aiming and shooting
  • Player movement including sprint, crouch, and jump
  • Multiple weapons with unique mobile-friendly mechanics
  • AI enemies optimized for mobile performance
  • Immersive Afghan-inspired environments
  • Work-in-progress demo available on mobile

Tech Stack

Unity Engine C# Mobile Touch Controls Unity Physics & AI

Challenges & Solutions

Developing for mobile devices required optimizing touch controls, FPS mechanics, and performance on various devices. I implemented object pooling, lightweight AI, and efficient camera control scripts to ensure smooth gameplay on smartphones and tablets.

Screenshots / Demo

Results & Impact

This project allowed me to deepen my Unity skills, develop mobile FPS mechanics, and optimize AI and controls for smartphones. It represents a milestone as one of the first Afghan mobile FPS games.

Future Improvements

Upcoming updates include more weapons, advanced AI, multiplayer mobile modes, optimized performance for low-end devices, and enhanced level designs. A playable demo will be released soon for Android and iOS.

Deep Delve

Uncover ancient pairs in the sands of time.

Project Overview

Deep Delve is a puzzle adventure that takes players into buried chambers beneath shifting sands. It combines memory and discovery as players match hidden artifact pairs concealed in ornate tomb walls. Each successful match collapses layers of dust and stone, unlocking deeper corridors, rare relics, and long-forgotten secrets.

Your Role

I was the sole developer for this project, responsible for designing the gameplay, coding the mechanics, creating the visuals, and optimizing performance across devices.

Features

  • Engaging memory puzzle gameplay with progressive levels
  • Immersive visuals inspired by ancient tombs and artifacts
  • Two different themed versions with unique design elements
  • Responsive design optimized for desktop and mobile devices
  • Smooth animations and intuitive user interface

Tech Stack

Unity Engine C# Responsive Design

Challenges & Solutions

One major challenge was optimizing asset loading for smooth performance on mobile devices. I addressed this by implementing lazy loading for textures and compressing image assets without sacrificing quality. Additionally, ensuring seamless transitions between levels required efficient memory management, which I solved by pooling objects and reusing resources.

Results & Impact

Deep Delve helped me improve my expertise in game development and UI design. Playtesting feedback showed strong engagement, with players enjoying the progressive difficulty and visual aesthetics. The project also served as a foundation for exploring more advanced puzzle mechanics in my future projects.

Future Improvements

Planned improvements include adding a scoring system, timed challenges, new tomb themes, and multiplayer cooperative modes where players can explore and solve puzzles together.

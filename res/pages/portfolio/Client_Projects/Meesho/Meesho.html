Meesho E-commerce Redesign

Complete UI/UX redesign of India’s leading online marketplace powered by modern front-end and seamless user experience, 2025.

Project Overview

In 2025, I was hired through Truelancer.com to lead the redesign of this website one of India’s largest e-commerce platforms. The goal was to create a faster, more visually engaging interface while improving usability and conversion rates across web and mobile devices. The new design focuses on modern typography, minimal navigation, and personalized shopping experiences.

Visit Website

My Role

I worked as a front-end designer and developer responsible for rebuilding the homepage, product discovery flow, and responsive layouts. My work included refactoring the previous UI into a modular system built with React components and integrating animations, accessibility improvements, and real-time API data.

Key Features

  • Fully responsive layout optimized for high mobile traffic
  • Dynamic product recommendations powered by ML APIs
  • Lightweight React components with smooth page transitions
  • Improved checkout experience with reduced drop-off rate
  • Dark mode UI support and SEO-friendly architecture

Tech Stack

React.js Next.js TypeScript TailwindCSS Node.js GraphQL Figma Google Cloud

Challenges & Solutions

Redesigning a high-traffic site like Meesho required balancing visual appeal with performance. Heavy images and scripts slowed the old version, so I implemented lazy loading, optimized assets, and leveraged Next.js server-side rendering. The team also used A/B testing to refine UX decisions based on live customer behavior data.

Results & Impact

The redesign launched successfully in Q3 2025, improving site performance by 38% and increasing user engagement by 22%. Mobile checkout completion rose significantly, thanks to the cleaner cart flow and optimized interface. The new Meesho design set a benchmark for accessible and inclusive e-commerce design in India.

Future Enhancements

Future updates will include integrating voice-based search, further personalization through AI, and adding community shopping features that allow users to share carts and wishlists in real-time.

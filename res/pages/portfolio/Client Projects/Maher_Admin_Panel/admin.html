Maher Bet Analyzer – Admin Panel

License management system built for my client to simplify selling and controlling app licenses.

Project Overview

This Admin Panel was built for Maher Bet Analyzer, an app that helps users predict winners in the TopTop betting game. The Analyzer app was license-based, and my client Maher sold licenses to users. Initially, all licenses were stored and managed directly in Firebase Realtime Database, but since my client was not familiar with Firebase/JSON, he struggled to maintain them. To solve this, I created a custom Admin Panel where he can easily add, update, and manage licenses without needing any technical knowledge.

Your Role

I designed and developed the Admin Panel from scratch. This included building the web UI in Flutter, connecting it to Firebase Realtime Database, and handling authentication for secure access. I also deployed it to Firebase Hosting so my client could access it anytime from a browser.

Features

  • Admin login with Firebase Authentication
  • Add, update, and delete user licenses
  • Real-time license validation and sync with Maher Bet Analyzer app
  • Search and filter licenses quickly
  • Simple and user-friendly UI for non-technical users
  • Deployed to Firebase Hosting

Tech Stack

Flutter (Web) Firebase Realtime Database Firebase Authentication Firebase Hosting Android Studio

Challenges & Solutions

The biggest challenge was simplifying license management for someone with no technical background. My solution was to build an intuitive dashboard that works just like a spreadsheet but directly connects to Firebase. Another challenge was separating Firebase projects (one for hosting, one for the Analyzer app). I solved this by using multiple Firebase configurations in the project.

Results & Impact

The Admin Panel made it effortless for my client to manage hundreds of user licenses. He no longer had to deal with raw JSON or Firebase directly, saving time and reducing errors. It also ensured smoother license distribution for Maher Bet Analyzer users, boosting customer satisfaction.

Future Improvements

Planned upgrades include usage analytics (to see which licenses are active), automated expiry notifications, and multi-admin support with role-based access.
